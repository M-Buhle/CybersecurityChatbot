# Cybersecurity Awareness Chatbot
## PROG6221 - Part 1
## Description
A command-line cybersecurity awareness chatbot that educates users about online safety.
## How to Run
1. Make sure you have .NET 9.0 installed
2. Clone this repository
3. Open terminal in the project folder
4. Type: dotnet run
5. Enter your name and start chatting!
## Features
- Voice greeting on startup
- ASCII art logo
- Personalised responses using your name
- Cybersecurity topics: passwords, phishing, safe browsing, malware, VPN, 2FA
- Input validation
- Coloured console interface
- Typing effect for conversational feel
